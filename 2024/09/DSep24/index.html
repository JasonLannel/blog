随忆·24年九月

发表于 2024-09-12 更新于 2024-10-08 分类于 水木录
大二了，写东西就随意随意啦。说白了就是懒惰了，摸鱼并且沉迷派派……？等等，原来已经是大三老登了？

十一

各位朋友们我回来啦！好开心好开心好开心

先画一个坑，你看这个坑又大又圆。但是这次应该能好好填坑了，flag 会拔掉的。

感觉非常奇妙，原来上次写（几乎不间断的正儿八经的）日记是在一年前了啊……把博客修好其实没有多少工作量，大概就是把之前写的博文搬进重新装的 Hexo 里。猜想之前博客爆掉可能是 Node.js 版本或者 hexo/next 版本之间不兼容的问题吧……

大二确实忙，天天昏天黑地，现在进大三了，过了不能承受的苦日子的结果是现在什么也不想上，能摆就摆，不能摆也想办法摆。好像这个比较像平均大学生生活。好在上学期抢到台球课了，这学期台球就是我生活的希望！台球，我赞美你！

科英今天莫名队列补上了，听说任务量巨大，但是我这学期（可以）没有硬课……如果是像英语听说那样的任务量，倒也能接受（当时课前发一段四级录音，课上完型填空，我都是偷偷听十几遍把全文听写下来的，要不然根本写不出来。）。另一门补上的是英语散文创意写作，上了一次课感觉也许不太适合我这种没有创意的人（）故事我是写不出一点，何况感觉课上例子用词太过深奥，没有词典我真的要流好多冷汗。

十二

坚持碎碎念第二天。

今天去看小朋友和大朋友们的实践总结会，在座的评委里好像只有我没有出去实践其实主要是玩，很奇怪为什么被拉来当评委了，毕竟我实践经历也没那么丰富。可能就是单纯缺人了，正好我刚加入院里联席会，准备开始多做一些社工，哈哈。听实践会感觉大家都去好多地方玩了，做了很多有意义的事情。比如川上风云支队去四川实践过程中组织了支教活动，唉，最听不得讲煽情支教故事了，听着就要落泪（消失期间寒假里我去四川支教了一周，感触非常深）。还有一点让我非常触动的是院里的总结会大家基本都在认真反思，实践内容有时候感觉甚至靠边站了，有点不太利于我打分。听大家非常认真地总结经验，在谈到怎么把实践做好这件事情上大家畅所欲言，我作为一名非常认同“没有调查就没有发言权”的人，处在这样一个积极热烈的调查经验分享会里，感觉如鱼得水，私底下就建议组长把经验留档给以后的同学借鉴。希望这事能一直办下去吧。

昨天提到的两门补上的课，今天在纠结退掉哪门好，具体原因和昨天说的一样。有朋友问我为什么不两门都退掉呢？我原本想的是院里面要求春研必须“基本修完前五学期课程”，不修的话还有两学分没修；大半夜转念一想，没道理啊，就剩两学分，这也卡我？遂全退了，并邮件院里老师，先斩后奏，决定如果院里卡我就直接跑去对线，上大学以来最有骨气的一回。何况我现在春研兴趣也没这么大，要是真不出去转悠下学期就要选不够六学分了。

十三

坏了，突然发现前两天自己日期标错了，大笨蛋

小学期出去云游了一番（勉强算是吧），虽然我个人并不看好目前 AI 的发展态势（太热，太卷，其中不免会充斥着许多水分），但作为一个工具，AI 又确实是万金油，是当前这个时代下一个研究者必须了解、清楚的工具，这学期打算好好地更深入地了解 AI，学会怎么去做各种 learning。虽然除了督促自己学 AI，还有一部分是为了了解多媒体的情况；但总之是出于这样的目的，跑去听导师开的课了。本想着偷偷听的，毕竟没有实际选课（担心自己没有足够的时间把作业写明白）；但是去得太晚，还是坐前面了，显然被看到了哈哈哈哈哈哈。希望导师别误会我只旁听不写作业（）

晚上校友聚会，嘿嘿！这次校友聚会倒是没有之前第一次参加时那样拘束了，虽然内里的社恐属性还是没改；用一句话说，算是外热内冷吧……看着学弟学妹们愈发感觉自己是老登了，比当年竞赛退役时的感觉还更加强烈。

是的，已经大三了，论交际来说，算是差强人意；但对于干劲，对于梦想，大概是不如学弟学妹们那般强烈了。我有生以来，还未经历过这样大的迷茫与不自信；我还不清楚自己该往何处去，但假若其他人告诉我应该往何处，我又有自己的坚持，或者是某种偏见。不知道从古至今的研究者们有多少人是这样的迷茫。

当然，这不影响我们学长学姐们给后辈们倾囊相授，这是两码事。自从我上大学以来，倘若让我向高中的后辈们介绍经验，我都会说很多很多话。关于未来的话，我也和他们说了许多；特别是希望他们在离高考还有些许时日的同时，要多去了解各个专业，深入地了解尚不能做到，但无论如何一定要早做一些预备与打算。不过，对于支教时那些处在县里、乡里的孩子们，甚至于城市里的一些孩子，他们又有否这样的条件，或者说，了解的渠道呢？又是否明白这样的重要性呢？更一般地，怎么去创造这样的条件，怎么让更多的后辈们更具象地认识到职业选择的重要性，为未来做打算呢？等待解答所需的时日，想必还有非常非常长吧。

十五

没做啥事，摸鱼的一天。最近经常打哈欠，非常困，大概是我阴间的作息导致的。还是得多睡一点。

十六

难得睡一个大懒觉。起来发现小孩拿了街霸法国第二，挺出乎意料。

最近一直在翻老东西，比如重新开始捡起音游，诸如邦邦（是的，我是0.25个邦邦人，但是pastel palettes）和 osu!；开始补一些老番，特指幸运星；今晚在看《列宁在一九一八》。过去的许多东西，十年前的，二十年前的，一世纪以前甚至更加久远的东西，较于现在就像是长视频与短视频的区别。长视频，对我而言，许多时候常常觉得浪费时间，于是便投身于短视频的海洋里；但偶尔翻开长视频，倒又觉得非常有魅力。这种魅力，不是画面上的，或者工艺上的，在这方面，科技的进步早已经把它们甩在时代奔驰的马车之后；我想可能更多是一种态度上的魅力：比如演员的态度，staff们的态度，等等。总觉得人物和人物之间，特别是往往不为人关注的许多细节上，有着宛若鸿沟的差别。现时代的节奏容不下叫好不叫座的作品；那些叫座的作品，从简单的概率出发，往往又经不起叫好。大概这样说的偏颇了点，但中秋还得早起去参加聚会，就不写太多了。我自己也得坚持多写点长视频这样的东西，有时还是得抬头看，一直低着头是看不见满月的。

此时已经是中秋凌晨二点，祝各位，也同祝我中秋快乐。

十九

去上早八射击课啦！第一次打气手枪，手抖得厉害，一抖了击发的时候就容易飞到一环两环去（）感觉站位应该挺重要的，第三轮站得靠右了些，子弹全偏左了……打得最好的第二轮打了两个十环三个九环，只飘了两个六环，可喜可贺。挺好玩，就是有点困。感觉最近非常疲倦，不知道是不是作息有点诡异的问题，从早到晚基本都在呼呼大睡，每次清醒时间不超过三个小时，之后一定要早睡早起（）

科研之前和我做一个项目的和我一届的学长（准确无误）似乎单干别的课题去了，原来项目的重担都交给我了。对于我而言倒是一件好事，免得天天摸鱼；让我现在比较犹豫的是做图形学的我应不应该出去春研，如果出去去什么地方找什么导师比较好。（打个广告，我现在非常愿意做免费劳动力，欢迎来抓我还可以看到我写的丑陋CV。）

二十、二十一

感冒了，勿念。

冷得太快，实在是不适应，有种瞬间从夏天到入冬的感觉，温差巨大。但是盖上被子又觉得热，怎么回事呢。

偶尔推敲

在次世代的昵称改成“栀知”啦，契机是想改一系列昵称防止通过 github 开盒。这个名字最开始是从“吱吱未央”来的，对应的是我干的职位——组织委员。吱吱听起来挺好记的，而且我今天想了想，听起来有点像 chi-chih（这是可以说的吗）。不过直接叫吱吱感觉太单调了，以前写小诗的人是这样的，所以另找了两个同音字。“栀”是想到栀子花了，以前还喜欢琢磨花的时候记下来的。（话说我在查花语的时候发现这种花明明有着坚韧、喜悦等很好的花语但是在风水里似乎不太吉利，你们这些民间传说要不掐一架？栀子还可以用药，这下更合我意了）“知”倒是挺朴素的，就是知道，知晓，连起来就是类似于“栀子（花）知道了”的意思。刚开始是觉得倒装比较有新意个鬼，没有用“知栀”；现在我想了想，“知栀”的主语在我，我知道栀子（坚韧美丽）；“栀知”是倒过来，栀子知道，知道谁呢？能入它之眼的，应当是和它相近，或者说它看得上的人。也算是一种美好祝愿吧。

所以以后可以叫我“吱吱”了，叫“吱吱未央”也可以。

二十二、二十三

招生报销琐事一大堆，作为一个在院里联席会信息表上写兴趣爱好是摸鱼的人来说实在是晴天霹雳，天啊，求求这堆破事快结束吧TAT 这两天是非常困但总是没法睡个好觉，有点红A战斗续行的感觉了（）不过身体没有红A那么健康就是了。

二十二去参加次世代迎新会，作为次世代老登（两年整，算是老登了吗？）又新加了一些群。这次混进组织部了，要有机会想多去干点活来着，毕竟看了这么久校园乐队，次世代已经算是在华子的二次元温暖小家，给家里边干活是应该的。顺便看看黑幕。今年迎新会整活挺多的，各个部门出了人设图，还有精彩现场wota艺，唯一可惜的是没有把握住今年的一站到底，今年题目挺有机会进决赛的，可惜自己脑子抽了倒在了鹿乃子乃子虎视眈眈上……

二十四、二十五

最近精神状态堪忧，周二上课有点心不在焉，写完计网作业后周三突然助教告知之前学堂上公示的课本版本是错的……在和同学的打印机和电子报销材料爆了一下午之后发现自己许多时间付之东流了，又生气又好笑又无奈又无力，唉，相较而言还是看图形学比较幸福。希望这辈子不要再一次性处理这么多报销材料了。这周伊始左肩膀到脖子好像是扯伤了，扭头非常困难，抬手都隐隐在痛，希望早日康复吧。

明天希望打枪打个好成绩，释放一下压力（）国庆快点来吧！
